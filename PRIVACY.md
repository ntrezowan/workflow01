# Privacy Policy

Workflow01 stores workspace data locally in Firefox extension storage on the user device.

Workflow01 does not collect, transmit, sell, share, or remotely process browsing data, workspace names, tab URLs, tab titles, or usage analytics.

Workflow01 does not use remote code, external services, analytics, telemetry, advertising, or tracking.

The manifest declares no data collection:

```json
"data_collection_permissions": {
  "required": ["none"]
}
```
